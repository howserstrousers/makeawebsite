launchinga.website

When you're launchinga.website don't forget to:

Add in Google Analytics code

Get the domain log-in credentials

Have a hosting provider like Heroku, Linode, Digital Ocean, etc.

Make sure you have webfont license(s)

Get an SSL certificate (Shopify is self-hosted and provides)

Email provider (like MailChimp) credentials (if you’re doing a submit / contact form)

Cross browser QA (Chrome, Safari, Firefox)

Cross device QA (iPhone 5, Andriod, iPad mini landscape mode)

‘Please upgrade your browser’ with a link to browsehappy.com for older browser versions

Add a ‘Please upgrade your browser’ screen with a link to browsehappy.com for older browser versions

Make a cool 404 error page with a .gif
